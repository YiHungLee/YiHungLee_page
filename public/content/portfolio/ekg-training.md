# 急救心律判讀訓練器

![|700x437](assets/file-20260214162956.webp)
## [點此開始遊玩](https://ekg.yi-hung-lee.work/)

【急救心律判讀訓練器】是一款專為醫護人員與急救學員設計的高壓模擬網頁遊戲。透過模擬真實的急救場景（Code Blue），訓練玩家在極短的黃金時間內判讀心電圖波形，並執行正確的電擊、CPR 或給藥決策。

## 專案介紹

這是一個基於原生 JavaScript 與 SVG 動態繪圖技術開發的臨床教學小遊戲。

專案旨在透過遊戲化機制，強化學員對致死性心律的反應速度與處置邏輯。

玩家將扮演急救現場的操作員，面對隨機生成的病患情境。系統會模擬生理監視器介面，即時顯示動態心電圖（ECG）與生命徵象（HR, SpO2）。

遊戲流程包含兩大核心挑戰：

1. **心律即時判讀**：玩家需在數秒內辨識出 NSR（正常竇性心律）、VF（心室顫動）、pVT（無脈性心室頻脈）、Asystole（心律停止）或 PEA（無脈性電氣活動）。
    
2. **黃金時間決策**：根據判讀結果，選擇按下「電擊」、「CPR」或「給藥」。系統設有嚴格的防呆與懲罰機制，例如對不可電擊的心律（如 Asystole）執行電擊將直接導致遊戲失敗。
    

# 功能亮點

- **動態波形演算法**：透過程式數學函數即時繪製 ECG 波形，模擬真實儀器的掃描線與雜訊，讓每次的心律表現都獨一無二。
    
- **沉浸式回饋體驗**：結合螢幕震動特效、心跳聲與警示音，以及倒數計時壓力條，營造急救現場分秒必爭的緊張氛圍。
    
- **有趣的警世結局系統**：本作最大特色在於失敗畫面的設計。若玩家處置失當（如對正常人電擊、過度 CPR 導致肋骨斷裂），遊戲結束時不會只顯示 Game Over，而是會收到一張詳細的**「法院刑事傳票」**，列出具體的醫療疏失與判決理由，深刻提醒臨床決策的嚴肅性。
